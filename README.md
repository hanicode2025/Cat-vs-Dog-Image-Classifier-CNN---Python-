# Cat vs Dog Image Classifier (CNN - Python)

**Overview:**  
Deep learning project that classifies images as cats or dogs using a Convolutional Neural Network (CNN). Demonstrates image preprocessing, data augmentation, model training, and evaluation.

**Technologies Used:**  
- Python 3.x  
- TensorFlow / Keras  
- NumPy  
- Matplotlib  

**Features:**  
- Image classification using CNN  
- Data augmentation (flip, zoom, shift)  
- Model training and validation  
- Performance evaluation on test data  

**Project Workflow:**  
1. Load and preprocess image dataset  
2. Apply data augmentation techniques  
3. Build and train CNN model  
4. Evaluate model performance  

**Key Skills Demonstrated:**  
- Deep learning with CNNs  
- Image preprocessing and augmentation  
- Model training and evaluation  
- Python-based ML workflows
